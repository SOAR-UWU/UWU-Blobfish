\doxysubsection{Detailed Description}
Implementation of a blinker for displaying errors on the Arduino.

The given LED (specified on initialisation) is blinked in a pattern specified in an int array (numbers are in ms). If another error occurs while midway through a blink pattern, the new error is ignored.

The blink functions are NON-\/\+BLOCKING. The function continue\+\_\+blinking has to be continuously called to ensure the LED is updated.
