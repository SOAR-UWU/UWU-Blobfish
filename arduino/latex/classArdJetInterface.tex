\doxysubsection{Detailed Description}
Implementation of the Arduino side of the Jetson-\/\+Arduino interface. 

Contains functions to establish a connection with the Jetson, and for exchanging of data.

Listens for motor values transmission over the Serial connection. 

An \textquotesingle{}M\textquotesingle{} character marks the start of a motor value message. If this function reads an initial character which is not \textquotesingle{}M\textquotesingle{}, it places the character received in the {\ttfamily putback} variable, for processing by other functions.

If the whole message is successfully received, this function returns true. Otherwise, this function returns false.
